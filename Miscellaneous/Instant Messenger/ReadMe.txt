This nifty program will alow you to send and recieve messages from other calculators if they are connected with USB cord, perfect if you want to 
send messages in class without talking or using a phone!

Instructions:
---------------------
~connect the two calculators, (you must have the program on both calculators to work, and the side of the cord labeled A needs to be on the sending calculator)
~open the program.
~press "compose" and type the message.
~press enter to send it
~while sending on one Calculator, press recieve on the other.
~Done! you can save the message, or continue texting, Have fun!
