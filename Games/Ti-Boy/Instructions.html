TI-Boy CE gameboy emulator

The TiBoy CE is a gameboy emulator that allows any gameboy programs to be installed onto the calculator however it must be noted that because gameboy programs are relatively large, you can't have too many installed.

Installation

For all apps, the TiBoy programs must be be installed:

  • TIBOYCE.8xp
  • TIBoyDat.8xv

An optional extra is TIBoySkn.8xv however I would recommend installing as it's small and it makes it look better

Conversion of a gameboy program

To begin with, you should download a gameboy program off a site such as Romsmania, etc. The Gameboy Color is the best one to go with as it has better looking games however games take a lot of the calculators space so there should only be one at a time. (Only delete the files you put on originally, new files are your saves)

Once this has been done, you should convert the file using the official Save converter
