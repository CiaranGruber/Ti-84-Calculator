## Keys:
    - ENTER = Pause/Play
    - UP/DOWN/LEFT/RIGHT = Move Player
    - MODE/APPS/ALPHA/STAT = Move Placer
    - X,T,0,n/DEL = Place/Delete Block
    - 2ND = Inventory
    - MATH/PRGM = Flying On/Off
    - VARS = Fight Mobs
    - 2ND = Choose Block & Exit (Inventory)
    - LEFT/RIGHT = Move Cursor (Inventory)
